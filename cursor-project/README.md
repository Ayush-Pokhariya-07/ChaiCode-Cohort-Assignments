# Cursor Landing Page Clone

A pixel-perfect recreation of the [Cursor.com](https://cursor.com) landing page, built entirely with **HTML** and **CSS**.

This project was built as an assignment for the **Chai Code Web Dev Cohort**. The goal was to master modern CSS layout techniques, specifically **CSS Grid** (Bento Grids) and **Flexbox**, while maintaining a high-quality "Dark Mode" aesthetic.

## What I Learned

- How to structure a complex landing page using semantic HTML.
- Mastering `display: grid` and `grid-column: span` to create irregular layouts.
- Using CSS variables for consistent color theming.
- Handling positioning (absolute/relative) for decorative elements like the "fake code window" dots.
